# 超级团队技能包

本目录包含超级团队模式的所有技能定义文件。

## 技能列表

| 技能文件 | 技能 ID | 说明 |
|---------|---------|------|
| `troubleshooting/SKILL.md` | troubleshooting | 问题诊断与排障 |
| `frontend-dev/SKILL.md` | frontend-dev | 前端开发 |
| `backend-dev/SKILL.md` | backend-dev | 后端开发 |
| `ui-design/SKILL.md` | ui-design | UI/UX 设计 |
| `database/SKILL.md` | database | 数据库操作 |
| `testing/SKILL.md` | testing | 测试与质量保障 |
| `documentation/SKILL.md` | documentation | 文档编写 |
| `security/SKILL.md` | security | 安全与合规 |
| `code-review/SKILL.md` | code-review | 代码审查 |

## 使用方式

这些技能由超级团队 Agent 自动调用，也可以手动指定：

```
@skill:frontend-dev 修改登录页面样式
@skill:backend-dev 新增用户统计接口
@skill:troubleshooting 页面加载报错
```
