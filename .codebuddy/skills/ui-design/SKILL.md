# 🎯 Skill: UI/UX 设计 (UI Design)

> **技能 ID**: `ui-design`
> **触发关键词**: 界面、设计、布局、美化、用户体验、UX、交互、样式

## 1. 技能职责

- 设计页面布局结构
- 输出交互状态说明
- 确保移动端适配
- 保持设计系统一致性

## 2. 设计原则

### 产品调性

| 关键词 | 设计体现 |
|-------|---------|
| **温暖** | 暖色调、柔和渐变、圆角 |
| **治愈** | 舒缓的动效、温馨的空状态 |
| **简洁** | 信息层级清晰、留白充足 |
| **安静** | 不打扰、低对比度提示 |
| **专注** | 减少干扰元素、突出核心功能 |

### 禁忌

- ❌ 强烈抖动/闪烁动效
- ❌ 频繁弹窗轰炸
- ❌ 高饱和度大面积色块
- ❌ 密集信息堆砌
- ❌ 社交压力元素

## 3. 设计系统

### 色彩规范

```css
--primary-color: #667eea;      /* 主色 - 紫蓝 */
--success-color: #10b981;      /* 成功 - 绿 */
--warning-color: #f59e0b;      /* 警告 - 橙 */
--error-color: #ef4444;        /* 错误 - 红 */
--text-primary: #1f2937;       /* 主文字 */
--text-secondary: #6b7280;     /* 次文字 */
```

### 间距规范

```css
--space-1: 4px;  --space-2: 8px;  --space-3: 12px;
--space-4: 16px; --space-5: 20px; --space-6: 24px;
```

### 圆角规范

```css
--radius-sm: 4px;   --radius-md: 8px;
--radius-lg: 12px;  --radius-xl: 16px;
```

## 4. 交互状态

| 状态 | 说明 | 视觉表现 |
|------|------|---------|
| **默认** | 正常展示 | 基础样式 |
| **悬浮** | 鼠标移入 | 轻微高亮/阴影增强 |
| **按下** | 点击瞬间 | 颜色加深/缩放 |
| **禁用** | 不可操作 | 灰色/降低透明度 |
| **加载** | 等待响应 | 旋转图标/骨架屏 |
| **空态** | 无数据 | 插图 + 文字提示 |
| **错误** | 操作失败 | 红色提示 + 重试 |

## 5. 移动端适配

| 项目 | 要求 |
|------|------|
| **触控区** | ≥ 44px × 44px |
| **字体** | ≥ 14px |
| **底部安全区** | padding-bottom: env(safe-area-inset-bottom) |

## 6. 设计检查清单

- [ ] 符合产品调性（温暖/治愈/简洁）
- [ ] 信息层级清晰
- [ ] 交互状态完整
- [ ] 移动端触控区 ≥ 44px
- [ ] 空状态设计
- [ ] 加载状态设计
- [ ] 错误状态设计
