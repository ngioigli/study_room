# UI/UX 设计 (UI Design)

设计页面布局、交互状态、确保移动端适配。

## 设计原则

### 产品调性

| 关键词 | 设计体现 |
|-------|---------|
| 温暖 | 暖色调、柔和渐变、圆角 |
| 治愈 | 舒缓动效、温馨空状态 |
| 简洁 | 信息层级清晰、留白充足 |
| 安静 | 不打扰、低对比度提示 |
| 专注 | 减少干扰、突出核心功能 |

### 禁忌

- ❌ 强烈抖动/闪烁动效
- ❌ 频繁弹窗轰炸
- ❌ 高饱和度大面积色块
- ❌ 密集信息堆砌

## 设计系统

### 色彩

```css
--primary-color: #667eea;      /* 主色 */
--success-color: #10b981;      /* 成功 */
--warning-color: #f59e0b;      /* 警告 */
--error-color: #ef4444;        /* 错误 */
--text-primary: #1f2937;       /* 主文字 */
--text-secondary: #6b7280;     /* 次文字 */
```

### 间距

```css
--space-1: 4px;  --space-2: 8px;  --space-3: 12px;
--space-4: 16px; --space-5: 20px; --space-6: 24px;
```

## 交互状态

| 状态 | 视觉表现 |
|------|---------|
| 默认 | 基础样式 |
| 悬浮 | 轻微高亮/阴影增强 |
| 按下 | 颜色加深/缩放 |
| 禁用 | 灰色/降低透明度 |
| 加载 | 旋转图标/骨架屏 |
| 空态 | 插图 + 文字提示 |

## 移动端适配

| 项目 | 要求 |
|------|------|
| 触控区 | ≥ 44px × 44px |
| 字体 | ≥ 14px |
| 底部安全区 | `padding-bottom: env(safe-area-inset-bottom)` |
