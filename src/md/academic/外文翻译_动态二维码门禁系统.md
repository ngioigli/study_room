# 外文翻译：基于动态二维码与物联网架构的高效门禁控制系统

**原文题目**：Efficient Access Control System Based on Dynamic QR Code and IoT Architecture  
**原文作者**：S. Kumar, et al. (模拟学术文献)  
**来源**：International Journal of Smart Infrastructure and IoT (2024)

---

## 1. 英文原文 (English Original)

### Abstract
With the rapid development of smart buildings and shared spaces, traditional physical access control methods are being replaced by digital solutions. This paper presents an efficient access control system based on dynamic QR code technology and Internet of Things (IoT) architecture. The proposed system addresses the security vulnerabilities of static QR codes, such as unauthorized duplication and sharing, by implementing a time-sensitive dynamic token mechanism. We introduce an "Atomic Passing Window" algorithm that manages concurrent access requests and ensures exclusive resource locking in high-traffic scenarios. Experimental results demonstrate that the system significantly reduces access conflicts and enhances the overall security of unmanned management environments.

### 1. Introduction
Access control is a fundamental component of physical security in modern facilities. Traditional RFID cards and biometric systems, while effective, often involve high deployment costs or privacy concerns. QR codes have emerged as a cost-effective alternative due to their ease of generation and recognition by smartphones. However, the static nature of standard QR codes allows them to be easily captured and shared, leading to security breaches. To mitigate this, dynamic QR codes—which change at regular intervals—have become the preferred choice for secure environments like smart study rooms and automated offices.

### 2. System Architecture
The proposed architecture follows a four-layer model: User Terminal, Gateway Simulation, Business Service, and Data Storage.
- **User Terminal**: A mobile web interface that interacts with the user to request access.
- **Gateway Simulation**: A Python-based client that synchronizes real-time credentials from physical devices to the cloud server.
- **Business Service**: A Spring Boot backend that manages identity authentication and access tokens.
- **Data Storage**: A relational database (MySQL) that maintains the state of locks and access logs.

### 3. The Atomic Passing Window Algorithm
A critical challenge in unmanned access management is the "Race Condition" where multiple users attempt to scan the door simultaneously. We propose the **Atomic Passing Window (APW)** algorithm. When a user requests to lock the QR code, the server checks the current state of the resource. If the resource is idle, a 180-second exclusive window is granted. During this window, the database state is updated atomically, and any subsequent requests from other users are queued or rejected with a "Channel Occupied" notification. This ensures that only one person can pass through the gate during a single recognition cycle.

## 2. 中文译文 (Chinese Translation)

### 摘要
随着智能建筑和共享空间的快速发展，传统的物理门禁控制方法正逐渐被数字方案所取代。本文提出了一种基于动态二维码技术和物联网（IoT）架构的高效门禁控制系统。所提出的系统通过实现时敏动态令牌机制，解决了静态二维码易被非法复制和分享的安全漏洞。我们引入了一种“原子化通行窗口”算法，用于管理并发访问请求，并确保在高流量场景下的资源排他性锁定。实验结果表明，该系统显著减少了访问冲突，并增强了无人值守管理环境的整体安全性。

### 1. 引言
门禁控制是现代设施物理安全的基础组成部分。传统的 RFID 卡和生物识别系统虽然有效，但往往涉及高昂的部署成本或隐私担忧。二维码由于易于生成且可被智能手机识别，已成为一种极具成本效益的替代方案。然而，标准二维码的静态特性使其容易被捕捉和分享，从而导致安全漏洞。为了缓解这一问题，定期更换的动态二维码已成为智能自习室和自动化办公室等安全环境的首选。

### 2. 系统架构
所提出的架构遵循四层模型：用户终端、网关模拟、业务服务和数据存储。
- **用户终端**：与用户交互以请求访问的移动端 Web 界面。
- **网关模拟**：基于 Python 的客户端，负责将物理设备的实时凭证同步到云端服务器。
- **业务服务**：基于 Spring Boot 的后端，管理身份验证和访问令牌。
- **数据存储**：关系型数据库（MySQL），维护锁定状态和访问日志。

### 3. 原子化通行窗口算法
无人值守门禁管理中的一个关键挑战是“竞态条件”，即多名用户尝试同时扫码。我们提出了**原子化通行窗口 (APW)** 算法。当用户请求锁定二维码时，服务器会检查资源的当前状态。如果资源处于空闲状态，则授予 180 秒的排他窗口。在此窗口期间，数据库状态会进行原子更新，来自其他用户的任何后续请求都将被排队或以“通道占用”通知予以拒绝。这确保了在单个识别周期内只有一个人能通过闸机。
